# 08 - WLAN 无线网络技术

## 知识点总结

### Wi-Fi 标准对照表

| 名称 | IEEE标准 | 最大速率 | 关键技术 |
|------|----------|----------|----------|
| Wi-Fi 4 | 802.11n | 600Mbps | MIMO(4空间流), 40MHz |
| Wi-Fi 5 | 802.11ac | 3.5Gbps | MU-MIMO(下行), 256-QAM |
| Wi-Fi 6 | 802.11ax | 9.6Gbps | OFDMA, 1024-QAM, TWT, BSS Color |
| Wi-Fi 7 | 802.11be | 46Gbps | 4096-QAM, 320MHz |

### Wi-Fi 6 相比 Wi-Fi 5 的提升

| 维度 | Wi-Fi 5 | Wi-Fi 6 |
|------|---------|---------|
| 调制 | 256-QAM | **1024-QAM**（效率+25%） |
| 多址 | OFDM单用户 | **OFDMA** 多用户并行 |
| MIMO | 下行MU-MIMO(4用户) | **上下行MU-MIMO**(8用户) |
| 节能 | 无 | **TWT** 目标唤醒时间 |
| 信道 | 80/160MHz | 160MHz |
| 速率 | 3.5Gbps | **9.6Gbps** |

### CAPWAP协议

```
AC (无线控制器)
  ├── 控制隧道 (UDP 5246): 配置下发、状态上报
  └── 数据隧道 (UDP 5247): 用户业务数据转发
        ↓
    FIT AP (瘦AP)
```

**核心功能**:
- AP发现与注册
- 配置下发（SSID、射频、安全策略）
- 数据转发
- 状态监控
- 固件升级

**建立阶段**: 发现 → 加入 → 配置 → 数据隧道建立

### WLAN安全

| 安全机制 | 特点 |
|----------|------|
| WEP | 开放系统认证，已被淘汰 |
| WPA2-PSK | 预共享密钥，AES加密 |
| WPA2-Enterprise | 802.1X认证，RADIUS |
| WPA3-SAE | SAE替代PSK，防暴力破解 |
| WPA3-Personal | 公共网络个性化加密 |

### WLAN分类

| 类型 | 覆盖范围 | 示例 |
|------|----------|------|
| WPAN | ~10米 | 蓝牙、ZigBee |
| WLAN | ~100米 | Wi-Fi |
| WMAN | 几公里~几十公里 | WiMAX |
| WWAN | 几十~上百公里 | 4G/5G |

### 隐藏节点问题与RTS/CTS

```
A ←→ B ←→ C
A、C互不可见，同时发数据给B → 碰撞

解决: RTS/CTS机制
1. A发RTS给B，申请信道
2. B广播CTS，通知其他节点暂停
3. C收到CTS后延迟发送
4. A完成传输后C再发送
```

### Wi-Fi 6 关键技术详解

**OFDMA（正交频分多址）**:
- 将信道划分为多个正交子载波
- 不同用户分配不同子载波
- 多用户并行传输，低时延

**TWT（目标唤醒时间）**:
- 终端不传数据时休眠
- 按需唤醒，延长电池续航
- 物联网设备特别受益

**BSS Color**:
- 给不同BSS着色标记
- 区分同频干扰，提升密集环境性能

## 常见问题与排错

| 问题 | 原因 | 解决 |
|------|------|------|
| 信号干扰 | 微波炉/蓝牙2.4GHz | 切换5GHz频段 |
| 连接不上AP | 密码错误/认证失败 | 检查安全配置 |
| 漫游断线 | 切换时延>50ms | 启用快速漫游(802.11r) |
| 网速慢 | 同频干扰 | 调整信道或启用BSS Color |

## 速记口诀

> **Wi-Fi六OFDMA多用户，1024-QAM速率高，TWT省电CAPWAP管，WPA3防破SAE强。**
