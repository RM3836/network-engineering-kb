# 10 - 网络爬虫技术 (Python)

## 知识点总结

### 核心概念
- **网络爬虫**: 自动抓取网页数据的程序
- **API接口**: 直接请求JSON数据，比解析HTML更高效
- **反爬机制**: Cookie验证、频率限制、User-Agent检测
- **数据清洗**: 从原始数据中提取有用信息

### 技术栈
```
requests (HTTP请求)
  ↓
json/re (数据解析)
  ↓
pandas (数据处理)
  ↓
matplotlib/wordcloud (可视化)
```

## 实操步骤

### 1. 微博数据爬取（移动端API）

```python
import requests
import pandas as pd
import time
import random
import re

# 配置
COOKIE = '从浏览器m.weibo.cn获取'
BASE_URL = 'https://m.weibo.cn/api/container/getIndex'
HEADERS = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
    'Cookie': COOKIE,
    'Referer': 'https://m.weibo.cn/',
    'X-Requested-With': 'XMLHttpRequest'
}

def crawl_page(page_num, keyword='机票'):
    """爬取单页数据"""
    params = {
        'containerid': f'100103type=1&q={keyword}',
        'page_type': 'searchall',
        'page': page_num
    }
    time.sleep(random.uniform(1, 3))  # 随机延迟，反反爬
    try:
        resp = requests.get(BASE_URL, params=params, headers=HEADERS, timeout=10)
        resp.raise_for_status()
        return resp.json()
    except Exception as e:
        print(f"第{page_num}页失败: {e}")
        return None

def parse_data(json_data):
    """解析JSON数据"""
    results = []
    cards = json_data.get('data', {}).get('cards', [])
    for card in cards:
        if card.get('card_type') == 9:  # 微博卡片
            mblog = card.get('mblog', {})
            results.append({
                '用户名': mblog.get('user', {}).get('screen_name', ''),
                '文本': re.sub(r'<[^>]+>', '', mblog.get('text', '')),
                '点赞数': mblog.get('attitudes_count', 0),
                '评论数': mblog.get('comments_count', 0),
                '转发数': mblog.get('reposts_count', 0),
            })
    return results

# 主程序
all_data = []
for page in range(1, 21):
    print(f"爬取第{page}页...")
    data = crawl_page(page)
    if data:
        all_data.extend(parse_data(data))

df = pd.DataFrame(all_data)
df.to_csv('weibo_data.csv', index=False, encoding='utf-8-sig')
print(f"共爬取 {len(df)} 条数据")
```

## 反爬对策

| 反爬手段 | 应对方法 |
|----------|----------|
| Cookie验证 | 从浏览器复制有效Cookie |
| 频率限制 | `time.sleep(random.uniform(1,3))` 随机延迟 |
| User-Agent检测 | 伪装浏览器UA |
| IP封禁 | 使用代理IP池 |
| 验证码 | 降低频率 / 打码平台 |

## 常见问题与排错

| 问题 | 原因 | 解决 |
|------|------|------|
| 返回空数据 | Cookie过期 | 重新从浏览器获取 |
| 403 Forbidden | 被反爬拦截 | 检查Headers+降低频率 |
| 编码乱码 | UTF-8/GBK混用 | `encoding='utf-8-sig'` |
| 数据不全 | JSON结构变化 | 检查API返回结构 |
